# 💳 Credit Card Fraud Detection

A machine learning project to detect fraudulent credit card transactions using a Logistic Regression model. The project includes data analysis, model building, and a Flask-based web application for real-time predictions.

---

## 🚀 Overview

Credit card fraud poses a significant threat in digital finance. This project uses machine learning to differentiate between genuine and fraudulent transactions based on anonymized transaction data.

---

## 🛠️ Tech Stack

- **Python**
- **Pandas, NumPy, scikit-learn** for data processing & modeling
- **Flask** for deploying the model as a web app
- **HTML/CSS** (in `templates/index.html`) for frontend (assumed)
